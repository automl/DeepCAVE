deepcave.utils.run_caches¶

Classes

RunCaches(config)

Holds the caches for the selected runs.
class deepcave.utils.run_caches.RunCaches(config)[source]¶

Bases: object

Holds the caches for the selected runs. The caches are used for the plugins to store the raw outputs so that raw outputs must not be calculated again.

Each input has its own cache. This change was necessary because it ensures that not all data are loaded if they are not needed.
